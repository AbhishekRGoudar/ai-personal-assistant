# AI Personal Assistant

A Python-based AI assistant that performs tasks using voice commands.

---

## Features
- Voice command recognition
- Open websites
- Automation tasks
- Text-to-speech responses

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Usage

Run the assistant:

```bash
python assistant.py
```

---

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Automation libraries
